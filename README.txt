For execute this application, you should get appengine_django, django.zip, .google_appengine first.

appengine_django: download from http://code.google.com/p/google-app-engine-django/
.google_appengine: download from http://code.google.com/appengine/downloads.html
django.zip: download from www.djangoproject.com and zip it by http://code.google.com/appengine/articles/django10_zipimport.html

After prepare all archives, put them near the app.yaml file.
